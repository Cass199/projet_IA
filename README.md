# Job Interview Application

This is a simple web application for job interview applications. It allows users to fill out a form to submit their job interview details and displays the results dynamically.

## Project Structure

```
job-interview-app
├── index.html       # Main HTML document
├── style.css        # Styles for the application
├── script.js        # JavaScript code for interactivity
└── README.md        # Project documentation
```

## Getting Started

To run this application locally, follow these steps:

1. **Clone the repository** (if applicable) or download the project files.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser.

## Features

- A user-friendly form for submitting job interview applications.
- Dynamic display of submitted information.
- Basic styling for a clean and professional look.
